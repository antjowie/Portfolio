Last Resort

2020 - 32 weeks - 23 students - network developer - BUas Y3 project

Last Resort is a co-op first person shooter featuring roguelite elements. Set on near-future Mars, you and up to 3 others gun down hordes of alien monstrosities in humankind's first extraterrestrial luxury resort. Unlock new weapons and upgrade them during runs to suit your playstyle.


    My contributions:

  • Profiling and optimizing network quality, guiding the team to create multiplayer supported features and code

  • Integrating netcode with Steam

  • Setting up first- and third person weapon based character animations

  • Multiplayer supported level streaming for a seamless game loop

Empires

2018 - 8 weeks - BUas Y2B project

A RTS game in which you have to colonize the entire galaxy, made for the Raspberry PI from scratch


    Features:

  • Remotely cross compiled on the Raspberry Pi 3 using Visual Studio 2017

  • Uses GLES 2 and supports primitive shapes, models and text

  • Supports ray tracing, allowing you to click planets and vessels which are represented in 3D

  • Procedurally generated galaxy with planets that can be colonized. Those then influence each other

  • Different vessel types with different behaviors (colonizers, fighters, merchants and carriers) and a TSP implementation for them to navigate

  • An AI to play against
