About the project

Temple of giants is my 2nd year project at BUas. It is the first project in which I worked with quite a number of students (16 to be exact). I have mostly been working on the player abilities with the VFX artist as well as general gameplay aspects such as respawning. This project has taught me a lot about working with students from other disciplines as well as the additional issues that come working with more students.

My contributions

Videos are placeholder

  • Implemented player abilities

    The main character uses abilities to solve the problem. The two main abilities that I worked on were creating platforms and creating projectiles which are used to activate switches. I have worked together with the VFX artist to create this feature.

  • Implemented the saving and checkpoint system

    Our levels consist of platforming sections. I have implemented saving and loading to the game. We also use this feature for some puzzles. Respawning is done via checkpoints. We save the last checkpoint so that the player can continue playing the game.

  • Implemented controller support (UI changes dynamically based on input)

    Together with the UI designer I worked on making the UI functional with controller input. Using the input system made with another programmer the game now recognizes which input device is used and updated the text and UI accordingly. All while the game is playing. This system has been inspired from A Hat in Time.

  • In game feedback system that forwards feedback to our Discord (or any webhook)

    Since we are working from home getting feedback has become a lot harder. At school, we would get a person in the hallway to playtest our game. Since we can't always see everyones screen anymore I created a system that allows us to send whatever is on our screen to our Discord channel. This system allowed us to gather feedback more easily, if people see something they like. They can send us a message about it. This system was inspired by Subnautica's method of gathering feedback.
